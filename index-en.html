What is FreeWnn?

  • 'Wnn' is pronounced 'un-nu'. Basic information of Wnn is described in (English version of) "What is Wnn?".
  • FreeWnn is derived (and GPL-ed) version of Wnn4.2, contributed X11 software.
  • FreeWnn is a client-server type Kana (pronounciation) - Kanji in Japanese translation system. It also converts to Hanzi in Chinese, both Simplified and Traditional, and Hanja in Korean.
  • Distributed under GNU GPLv2 / LGPLv2 (mainly library part) or later version of each licenses. (Some of the file may distribute separately under more loose license. See each file for details.)

Current (alpha) version: 1.1.1-a023

Currently, I (Aono) will not make changes to FreeWnn code. I imported (and converted) CVS repository in OSDN (hosted in m17.org in the past) to https://github.com/freewnn/FreeWnn-Archive .

NOTE: We don't officially have a client implementation now. (You can try uum ( terminal frontend for Japanese) by adding '--enable-client' with the configure script, but it is experimental.) Please use (for example) Tamago (for Emacs), kinput2, im-ja and other clients using libwnn.

Further Development

As I described, I will not make changes to FreeWnn code as of 2025. I will not receive pull-request to https://github.com/freewnn/FreeWnn-Archive for now.
